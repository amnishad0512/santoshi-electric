# Santoshielectric

Welcome to **Santoshielectric** – your one-stop online destination for buying and selling top-quality electronic products!

## 🔌 Overview

Santoshielectric is a modern eCommerce web platform built to make electronic shopping seamless, fast, and secure. Whether you're looking to purchase the latest gadgets or sell your electronics, this website provides a user-friendly experience with all essential eCommerce features.

## 🛍️ Key Features

- 🔎 **Product Browsing & Search** – Easily search and filter electronics by category, brand, and price.
- 🛒 **Shopping Cart & Checkout** – Add items to cart, apply discounts, and securely checkout.
- 💬 **User Authentication** – Secure login and signup with account dashboard for buyers and sellers.
- 🛠️ **Product Management** – Sellers can add, edit, and manage product listings.
- 💳 **Payment Integration** – Integrated with secure payment gateways for smooth transactions.
- 📱 **Responsive Design** – Optimized for mobile, tablet, and desktop experiences.

## 🧱 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Next.js
- **Backend:** Laravel
- **Database:** MySQL
- **Authentication:** Laravel Sanctum
- **Payments:** Razorpay
- **Hosting:** Hostinger
